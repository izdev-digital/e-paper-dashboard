# E-Paper Dashboard Packaging

Hardware enclosure designs for the E-Paper Dashboard device.

## Contents

This directory contains CAD files for 3D-printable enclosures and mounting solutions for the E-Paper Dashboard hardware.

## Supported Hardware

The enclosure is designed to house:
- Waveshare E-Paper ESP32 Driver Board
- Waveshare 7.5inch e-Paper HAT (B)

## Usage

1. Open the `.FCStd` file in [FreeCAD](https://www.freecad.org/)
2. Export to STL format for 3D printing
3. Print using your preferred 3D printer and material
4. Assemble the printed parts with the electronic components
